Kontakterfassung

Zu einer weniger für Fehler anfälligen Erfassung der Kontaktdaten könne man ein solches Formular nutzen. In diesem Beispiel werden werden keine Daten im Browser oder auf Servern gespeichert. Die Daten werden lediglich per E-Mail an einen Verantwortlichen zur nahtlosen Erstanmeldung erfasst.
